# Contributing to Linear Algebra Python

Thank you for your interest in improving this repository! This project is designed for learning and sharing Python-based linear algebra explorations.

## Getting Started

1. Fork the repository.
2. Clone your fork:
   ```bash
git clone https://github.com/<your-username>/linear-algebra-python.git
cd linear-algebra-python
```
3. Create and activate a virtual environment:
   ```bash
python -m venv .venv
source .venv/bin/activate
```
4. Install dependencies:
   ```bash
python -m pip install --upgrade pip
pip install -r requirements.txt
```

## How to Contribute

### Report a bug
Use the `Bug report` template in `.github/ISSUE_TEMPLATE/bug_report.md` and include:
- a short summary of the issue,
- steps to reproduce,
- expected vs actual behavior,
- your Python environment and package versions.

### Request a feature
Use the `Feature request` template in `.github/ISSUE_TEMPLATE/feature_request.md` and include:
- a clear description of the improvement,
- the benefit to the project,
- any implementation ideas.

### Submit a pull request
1. Create a branch for your change:
   ```bash
git checkout -b feature/your-description
```
2. Make your changes.
3. Run the existing scripts to verify output.
4. Commit with a clear message.
5. Push your branch and open a pull request.

### Code style
- Keep code simple and readable.
- Prefer descriptive names for scripts, functions, and variables.
- Document new features or scripts with comments or a short usage note.

## Notes

- The project currently focuses on Python scripts and generated visualizations.
- If you add new examples, please include a short description in the `README.md` or notes files.
