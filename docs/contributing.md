# Contributing

Contributions are welcome to improve the linear algebra learning experience.

See the repository root [`CONTRIBUTING.md`](https://github.com/venkatavikas69/linear-algebra-python/blob/main/CONTRIBUTING.md) for full contribution guidelines.

## Filing issues

- Use the bug report or feature request template in `.github/ISSUE_TEMPLATE/`.
- Provide a clear description, reproduction steps, and environment details.

## Pull requests

- Fork the repository and create a small, focused branch.
- Test your changes locally.
- Submit a pull request with a clear summary of what changed.
