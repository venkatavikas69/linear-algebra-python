# Linear Algebra Python

Welcome to the documentation site for the `linear-algebra-python` repository.

This project provides Python practice for core linear algebra concepts:

- vectors and plotting
- vector addition and scalar multiplication
- linear combinations, span, and basis vectors
- generated visualizations and study notes
